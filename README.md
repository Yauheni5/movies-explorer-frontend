### Использовано:

- React
- jsx
- express
- mongoDB
- Модальные окна, работа с формами(JS validation)
- работа с API
- Регистрация и авторизация пользователя
- Сборка проекта Webpack-ом
- улучшенный UX
- ООП
- методология БЭМ
- Flexbox
- Grid
- Адаптация под разные экраны устройств
- Проект выполнен согласно макета в Figma
